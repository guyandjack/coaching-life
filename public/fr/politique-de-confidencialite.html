Politique de Confidentialité

1. Introduction

Bienvenue sur "socoaching.ch", exploité par "SO COACHING", situé à Sion en Valais. Nous nous engageons à protéger la confidentialité de vos données personnelles. Cette politique de confidentialité explique quelles données nous collectons, comment nous les utilisons et les protégeons.

2. Données Collectées

Nous collectons les données personnelles suivantes :

  • Nom
  • Prénom
  • Adresse email
  • Autres informations
  • Adresse IP

3. Moyens de Collecte des Données

Nous collectons vos données personnelles par les moyens suivants :

  • Formulaires de contact
  • Script

4. Utilisation des Données Collectées

Les données collectées sont utilisées pour :

  • Fournir nos services de coaching et de consulting
  • Communiquer avec vous
  • Envoyer des informations promotionnelles et des newsletters
  • Améliorer notre site et nos services

5. Partage des Données Personnelles

Nous ne partageons vos données personnelles qu'avec :

  • Les autorités légales en cas d'obligation légale.

6. Durée de Conservation des Données

Nous conservons vos données personnelles aussi longtemps que nécessaire pour fournir nos services et respecter nos obligations légales.

7. Droits des Utilisateurs

Vous avez le droit de :

  • Accéder à vos données personnelles
  • Rectifier vos données personnelles
  • Demander la suppression de vos données personnelles
  • Vous opposer au traitement de vos données personnelles
  • Demander la portabilité de vos données personnelles

Pour exercer ces droits, veuillez nous contacter à sophiechevrolet@gmail.com.

8. Sécurité

Nous mettons en œuvre des mesures de sécurité techniques et organisationnelles pour protéger vos données personnelles contre tout accès non autorisé, perte ou divulgation.

9. Utilisation de Cookies

Nous utilisons uniquement des cookies strictement nécessaires pour améliorer votre expérience et les fonctionnalités du site.

Aucun cookie n'est utilisé pour connaître vos habitudes de navigation sur le web, ou pour vous fournir une publicité ciblée.

Votre adresse IP est obtenue via un script afin d'obtenir des informations sur la provenance des visiteurs. Cette IP est utile pour identifier, le cas échéant, un nouveau visiteur et ainsi augmenter la précision de nos statistiques liées aux visites de notre site. Le cheminement des visites sur les pages de notre site est associé à une adresse IP.

10. Modifications de la Politique de Confidentialité

Nous nous réservons le droit de modifier cette politique de confidentialité à tout moment. Les modifications seront effectives dès leur publication sur cette page.
